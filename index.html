📋 What You Can Do

With the Editor:

  1. Upload 360° panorama images
  2. Place interactive hotspots visually
  3. Configure navigation between scenes
  4. Customize hotspot colors and titles
  5. Export as JSON or standalone HTML

With the Viewer:

  1. Load exported tour JSON files
  2. Navigate immersive 360° tours
  3. Share tours without coding

📝 Workflow Example

  1. Open Editor (click button above)
  2. Click + Add Scene and upload panorama images
  3. Click + Add Hotspot then click on the preview to place
  4. Set hotspot target scene and customize properties
  5. Click Export button
  6. Choose Export Viewer HTML for standalone tour
  7. Share the exported HTML file - Done! 🎉

💡 Tips

  • Editor auto-saves to localStorage every 30 seconds
  • Use drag-and-drop to reorder scenes
  • Exported viewer HTML includes everything - no dependencies needed
  • JSON export is compatible with the SWT library for custom integration
  • For best results, use 4096x2048 equirectangular panoramas
